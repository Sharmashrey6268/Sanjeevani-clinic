# Child Development & Therapy Center Website

A production-ready, responsive, SEO-focused website for a pediatric occupational therapy center in Indore, India.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide icons

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Customize

Update business details in:

```txt
data/site.ts
```

Important fields:

- Clinic name
- Phone number
- WhatsApp number
- Address
- Google Maps link
- Therapist name, qualification, and experience
- Services and testimonials

## Deployment

Deploy directly on Vercel.

```bash
npm run build
```

## Notes

The contact form is frontend-only. Connect it later using:

- Next.js API route
- EmailJS
- Formspree
- Firebase
- Any CRM/webhook
